Iron Beads Editor Help

アイロンビーズエディタ Help

Iron Beads Editor(IBE)はあくまで「アイロンビーズ」のデータを作る用のエディタとして開発しました。
ドット絵を描く用には適していません。

Iron Beads Editor(IBE) was developed solely as an editor for creating "Iron beads" data.
It is NOT suitable for drawing dot pictures.

ショートカットキー

Short Cut Keys

  • 1キードット打ちモード
  • 2キー線画モード
  • 3キー塗りつぶしモード
  • 4キー消しゴムモード
  • Alt + 既存の部分をクリックスポイト機能
  • Rキー左右反転
  • Bキーミラー機能
  • Shift + スクロール拡大/縮小(左側のボタンより大きく拡大縮小します)
  • Alt + スクロール下絵の拡大/縮小(左側のボタンより大きく拡大縮小します)
  • Alt + 矢印キー下絵の移動(左側のボタンより大きく移動します)
  • Ctrl(Command) + ZキーUndo(前の動作に戻る
  • Ctrl(Command)+shift + ZキーRedo(次の動作に戻る)
  • 1A Dot
  • 2Straight line
  • 3Fill
  • 4Eraser
  • Alt + ClickDropper
  • RReverses the edited data left and right.
  • BMirror Mode(Fills in symmetrical areas.)
  • Shift + Mouse scroll ButtonZoom-In(out) dots width.
  • Alt + Mouse scroll ButtonZoom-In(out) background image.
  • Alt + Arrow keysMove background image.
  • Ctrl(Command) + ZUndo
  • Ctrl(Command)+shift + ZRedo
